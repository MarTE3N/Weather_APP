#Aplikacja pogodowa

### Installation

Get your api token from [openweathermap.org](https://openweathermap.org/price)
Clone the repo
   ```sh
   git clone https://github.com/MarTE3N/Weather_APP
   ```
Install packages
   ```sh
   yarn install
   ```
Enter your token in `config.js`
